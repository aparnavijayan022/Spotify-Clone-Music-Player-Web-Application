# 🎵 Spotify Clone

A responsive Spotify-inspired music player built with **HTML, CSS, JavaScript, and Bootstrap**. This project replicates the core user interface and essential music playback features while demonstrating responsive design and interactive frontend development.

## 🚀 Features
- Responsive Spotify-inspired UI
- Play/Pause, Next & Previous controls
- Interactive progress bar
- Playlist interface
- Modern and clean design

## 🛠️ Technologies
- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5

## ▶️ Getting Started
Clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/aparnavijayan022/Spotify-Clone-Music-Player-Web-Application.git
```
